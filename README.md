# Sahayakai

Eligibility prediction and assistance platform powered by AI.

## Project Structure

```
sahayakai/
├── frontend/          # React frontend application
├── docs/              # Documentation
├── assets/            # Project assets (logos, banners)
└── .github/           # GitHub workflows and configs
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```
